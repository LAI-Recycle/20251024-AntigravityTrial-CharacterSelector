# VTuber 伴侶轉盤

互動式 VTuber 主題的「命定角色」轉盤。使用者可在首頁透過卡片輪播瀏覽多位角色，點擊即可切換到詳細檔案，查看角色故事與屬性條。整體介面以玻璃擬態與柔和漸層打造出夢幻感，支援滑鼠、鍵盤與觸控操作。

## 功能特色

- **角色輪播**：CSS 3D 轉場搭配 `script.js` 的 `updateCarousel()`，顯示當前卡片與前後預覽卡。
- **詳細檔案**：`showProfile()` 切換到 `profile-view`，展示圖片、描述與三種屬性條並以 `requestAnimationFrame` 製造動畫。
- **多重操作方式**：左右鍵、Enter/Space、滑鼠拖曳與觸控滑動皆可切換卡片。
- **響應式設計**：`style.css` 針對 768px、600px、380px 等斷點調整版面，手機視圖採 Bottom Sheet 風格。
- **素材管理**：`assets/` 內含角色立繪，透過 `characters` 陣列載入。

## 快速開始

1. 下載或 clone 專案到任一資料夾。
2. 使用瀏覽器開啟 `index.html`（可直接雙擊或在終端輸入 `start index.html`）。
3. 在桌機或手機上體驗角色輪播並切換各個 VTuber 檔案。

> 本專案為純前端靜態頁面，不需額外安裝套件或執行建置指令。

## 專案結構

```
.
├── assets/          # 角色圖片
├── index.html       # 頁面骨架與主要 DOM 節點
├── style.css        # 主題色彩、輪播動效與響應式樣式
└── script.js        # 角色資料、事件綁定與檢視切換邏輯
```

## 可擴充方向

1. **資料來源 API 化**：改為從遠端 JSON 取得角色資料，方便後台更新。
2. **收藏或分享功能**：在詳細檔案增加按鈕，儲存或分享喜愛角色。
3. **動畫與音效**：切換角色時加入音效、粒子特效強化沉浸感。
4. **多語系支援**：以 `data-i18n` 或翻譯檔提供繁中/日文/英文切換。
